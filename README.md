# Лабораторна робота №2: Розробка специфікації API та коду програми для роботи у хмарному середовищі

Це простий REST API додаток, створений на Laravel, який імітує базовий функціонал платформи Q&A

## Специфікація API

### Отримати всі питання
Повертає список усіх доступних питань
* Метод: `GET`
* Ендпоінт: `/questions/`
* Успішна відповідь: `200 OK`

### Створити нове питання
Додає нове питання до системи
* Метод: `POST`
* Ендпоінт: `/questions/`
* Тіло запиту (JSON): `json
  {
    "authorId": 101,
    "title": "Заголовок питання",
    "body": "Текст питання"
  }`
* Успішна відповідь: `201 Created`

### Отримати відповіді на конкретне питання
Повертає масив відповідей, привʼязаних до конкретного ID питання
* Метод: `GET`
* Ендпоінт: `/questions/{id}/answers`
* Параметри URL: {id}
* Успішна відповідь: `200 OK`
